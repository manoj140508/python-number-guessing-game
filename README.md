# Python Number Guessing Game

A simple number guessing game built using Python.

## Description

The game contains four secret numbers:

- a = 32
- b = 47
- c = 58
- d = 79

The player chooses one of the letters (`a`, `b`, `c`, or `d`) and then tries to guess the hidden number.

The game provides hints:

- "Higher than your guess"

- "Lower than your guess"

- "Very close!" (within 5 of the correct number)

The game also counts the number of attempts needed to guess the number correctly.

## Features

- User input

- Conditional statements (`if`, `elif`, `else`)

- `while` loop

- Attempt counter

- Hint system

- Beginner-friendly code

## How to Run

1. Make sure Python is installed.

2. Download or clone this repository.

3. Run the program:

python guessing_game.py
